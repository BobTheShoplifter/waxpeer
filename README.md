[WaxPeer](https://waxpeer.com) API wrapper for Node.js

Full API documentation [here](https://api.waxpeer.com/docs)

## Installation 
```sh
$ npm install waxpeer
```

### Initialization
```typescript
import { Waxpeer } from 'waxpeer'

const WP = new Waxpeer(process.env.WAXPEER_API, key)
```
